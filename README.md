# 🏦 Bank Customer Churn Analysis

## 📌 Overview
End-to-end data analysis project identifying why bank 
customers churn using Python, SQL, ML and Power BI.

## 🎯 Key Findings
- Overall churn rate: **20.4%**
- Germany has highest churn: **32.44%**
- Ages 50-60 churn at: **56.21%**
- Top churn driver: **Age (32.8% importance)**
- Inactive members churn **2x more** than active ones
- German females highest risk segment: **40.74%**

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| Python | Data cleaning & EDA |
| Pandas | Data manipulation |
| Seaborn/Matplotlib | Visualizations |
| SQL/SQLite | Business queries |
| Scikit-learn | ML models |
| Power BI | Interactive dashboard |

## 🤖 ML Models Built
- Random Forest Classifier (~88% accuracy)
- Feature Importance Analysis
- K-Means Customer Segmentation (4 clusters)
